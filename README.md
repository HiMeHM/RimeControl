# RimeControl

## 说明

这个工具是[小狼毫 Weasel](https://rime.im/)的基本配置工具，使用WPF开发。  
这个工具最开始于2016初，是我自学C#的练习项目；之后我就忙着毕业、实习、工作就一直没时间管,非常抱歉。
2019年春节重写了一次。如果遇到问题或者有什么建议，可以在issues反馈或者发邮件到邮箱（<1396715343@qq.com>），工作之余我尽量改善，感谢支持！
